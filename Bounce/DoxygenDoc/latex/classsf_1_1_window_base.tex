\doxysubsection{Detailed Description}
\doxylink{classsf_1_1_window}{Window} that serves as a base for other windows. 

\doxylink{classsf_1_1_window_base}{sf\+::\+Window\+Base} serves as the base class for all Windows.

A \doxylink{classsf_1_1_window_base}{sf\+::\+Window\+Base} can create its own new window, or be embedded into an already existing control using the create(handle) function.

The \doxylink{classsf_1_1_window_base}{sf\+::\+Window\+Base} class provides a simple interface for manipulating the window\+: move, resize, show/hide, control mouse cursor, etc. It also provides event handling through its \doxylink{classsf_1_1_window_base_a6a143de089c8716bd42c38c781268f7f}{poll\+Event()} and \doxylink{classsf_1_1_window_base_aa1c100a69b5bc0c84e23a4652d51ac41}{wait\+Event()} functions.

Close the window and destroy all the attached resources. 

After calling this function, the \doxylink{classsf_1_1_window}{sf\+::\+Window} instance remains valid and you can call \doxylink{classsf_1_1_window_base_a3b888387b7bd4be38d6db1234c8d7ad4}{create()} to recreate the window. All other functions such as \doxylink{classsf_1_1_window_base_a6a143de089c8716bd42c38c781268f7f}{poll\+Event()} or display() will still work (i.\+e. you don\textquotesingle{}t have to test \doxylink{classsf_1_1_window_base_aa43559822564ef958dc664a90c57cba0}{is\+Open()} every time), and will have no effect on closed windows.
