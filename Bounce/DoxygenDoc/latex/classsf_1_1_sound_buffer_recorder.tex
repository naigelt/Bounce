\doxysubsection{Detailed Description}
Specialized \doxylink{classsf_1_1_sound_recorder}{Sound\+Recorder} which stores the captured audio data into a sound buffer. 

\doxylink{classsf_1_1_sound_buffer_recorder}{sf\+::\+Sound\+Buffer\+Recorder} allows to access a recorded sound through a \doxylink{classsf_1_1_sound_buffer}{sf\+::\+Sound\+Buffer}, so that it can be played, saved to a file, etc.

It has the same simple interface as its base class (\doxylink{classsf_1_1_sound_recorder_a715f0fd2f228c83d79aaedca562ae51f}{start()}, \doxylink{classsf_1_1_sound_recorder_a8d9c8346aa9aa409cfed4a1101159c4c}{stop()}) and adds a function to retrieve the recorded sound buffer (\doxylink{classsf_1_1_sound_buffer_recorder_a1befea2bfa3959ff6fabbf7e33cbc864}{get\+Buffer()}).

As usual, don\textquotesingle{}t forget to call the \doxylink{classsf_1_1_sound_recorder_aab2bd0fee9e48d6cfd449b1cb078ce5a}{is\+Available()} function before using this class (see \doxylink{classsf_1_1_sound_recorder}{sf\+::\+Sound\+Recorder} for more details about this).
