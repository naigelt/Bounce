\doxysubsection{Detailed Description}
\doxylink{classsf_1_1_window}{Window} that can serve as a target for 2D drawing. 

\doxylink{classsf_1_1_render_window}{sf\+::\+Render\+Window} is the main class of the Graphics module. It defines an OS window that can be painted using the other classes of the graphics module.

\doxylink{classsf_1_1_render_window}{sf\+::\+Render\+Window} is derived from \doxylink{classsf_1_1_window}{sf\+::\+Window}, thus it inherits all its features\+: events, window management, Open\+GL rendering, etc. See the documentation of \doxylink{classsf_1_1_window}{sf\+::\+Window} for a more complete description of all these features, as well as code examples.

On top of that, \doxylink{classsf_1_1_render_window}{sf\+::\+Render\+Window} adds more features related to 2D drawing with the graphics module (see its base class \doxylink{classsf_1_1_render_target}{sf\+::\+Render\+Target} for more details). Activate or deactivate the window as the current target for Open\+GL rendering. 

A window is active only on the current thread, if you want to make it active on another thread you have to deactivate it on the previous thread first if it was active. Only one window can be active on a thread at a time, thus the window previously active (if any) automatically gets deactivated. This is not to be confused with \doxylink{classsf_1_1_window_base_a448770d2372d8df0a1ad6b1c7cce3c89}{request\+Focus()}.
